# Google Docs Clone

Built using:

- Next.js
- React.js
- Draft.js
- Tailwind CSS
- Material UI
- Firebase, Firestore
- Next Auth

## Features and functionality

- Login with your google account
- Create new documents
- Edit exisiting documents
- Sign out
